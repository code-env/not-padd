Props let you pass data from one component to another, making your UI reusable and flexible.

Hereâ€™s a quick example:

```jsx
function Greeting({ name }) {
  return <h2>Hello, {name}!</h2>;
}

export default function App() {
  return (
    <div style={{ textAlign: "center", marginTop: "40px" }}>
      <Greeting name="Zenith" />
      <Greeting name="Romaric" />
    </div>
  );
}
```

### How It Works

- The `Greeting` component receives a `name` prop.

- Props are just arguments you pass to components.

- Each instance can display different data.

Props make components dynamic and help you avoid repeating code.
